Getting Started with VittoriaDB

Quick Installation

VittoriaDB can be installed in multiple ways:

Option 1: Pre-built Binaries

Download from GitHub releases and run immediately:

wget https://github.com/antonellof/VittoriaDB/releases/download/v0.1.0/vittoriadb-v0.1.0-linux-amd64.tar.gz
tar -xzf vittoriadb-v0.1.0-linux-amd64.tar.gz
./vittoriadb run

Option 2: From Source

Build from source with Go:

go install github.com/antonellof/VittoriaDB/cmd/vittoriadb@latest
vittoriadb run

First Steps

  1. Start the server: Run vittoriadb run
  2. Create a collection: Use the REST API to create your first collection
  3. Insert vectors: Add your vector data
  4. Search: Perform similarity searches

API Examples

Here are some basic API calls to get you started:

Create Collection

curl -X POST http://localhost:8080/collections \
  -H "Content-Type: application/json" \
  -d '{"name": "documents", "dimensions": 384}'

Insert Vector

curl -X POST http://localhost:8080/collections/documents/vectors \
  -H "Content-Type: application/json" \
  -d '{"id": "doc1", "vector": [0.1, 0.2, 0.3, ...], "metadata": {"title": "Test Document"}}'

Search

curl "http://localhost:8080/collections/documents/search?vector=0.1,0.2,0.3,...&limit=5"
