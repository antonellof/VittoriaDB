What is VittoriaDB?

VittoriaDB is a high-performance, embedded vector database designed specifically for local AI development and production deployments. It provides a zero-configuration solution for vector similarity search, making it perfect for RAG applications, semantic search, and recommendation systems.

Quick Start

Get started with VittoriaDB in just a few commands:

# 1. Download and run
./vittoriadb run

# 2. Create collection
curl -X POST http://localhost:8080/collections \
  -H "Content-Type: application/json" \
  -d '{"name": "documents", "dimensions": 384}'

# 3. Insert vector
curl -X POST http://localhost:8080/collections/documents/vectors \
  -H "Content-Type: application/json" \
  -d '{"id": "doc1", "vector": [0.1, 0.2, 0.3, ...], "metadata": {"title": "Sample Document"}}'

# 4. Search
curl "http://localhost:8080/collections/documents/search?vector=0.1,0.2,0.3,...&limit=5"

Document Processing

VittoriaDB can process various document formats and automatically chunk them for vector storage:

  • Plain Text (.txt) - Direct text processing
  • Markdown (.md) - With frontmatter support
  • HTML (.html) - Tag stripping and metadata extraction
  • PDF (.pdf) - Text extraction (requires library integration)
  • Word (.docx) - Document processing (requires library integration)
